# Birthday Wishes Website

A simple web application to generate personalized birthday wishes.

## Features

- Input a name to generate a random birthday wish
- Clean and responsive design
- Easy to share and deploy

## How to Use

1. Open `index.html` in your web browser
2. Enter the birthday person's name in the input field
3. Click "Generate Wish" to see a personalized message

## Files

- `index.html` - The main HTML structure
- `style.css` - Styles for the website
- `script.js` - JavaScript for generating wishes
- `README.md` - Project documentation

## Deployment

Host this on GitHub Pages:
1. Create a public GitHub repository
2. Upload all files to the repository
3. Go to Settings → Pages
4. Select `main` branch as source
5. Your site will be live at `https://yourusername.github.io/happy-birthday/`
